# Review Analyzer — MVP v1

## Goal
Paste or upload reviews → get themes + complaints + praises + fix-first list.

## Pages
- /
- /login
- /dashboard
- /projects/[id]

## MVP features
1) Email/password login
2) Create a Project
3) Add reviews (paste text + CSV upload)
4) Analyze reviews (placeholder logic for now)
5) Show results + save analysis history

## Not in v1
- Integrations (Shopify/Amazon/G2/Google)
- Scraping
- Team accounts
- Payments/Stripe
- Fancy charts
